This files purpose is as a styling guide for base HTML5 used in common scenarios and act as a reference for common elements used in a project.
If you are like me I don't like modifing a libraries css. I prefer duplicating a libraries style in my own 'override' file and change its styling there. I may even have independant override files named after the libraries they are overrideing.
This allows me to update libraries usually with little css issues. To use, reference all library css files first, then your custom 'override' css file. Once you have your initial external librarires referenced.
